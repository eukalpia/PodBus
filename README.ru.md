# PodBus

PodBus — транспортно-ориентированный набор инструментов для событий, request/reply и фоновых задач в Dart и Serverpod.

Основная документация находится в [README.md](README.md). Там есть таблица возможностей транспортов, быстрый старт, примеры JetStream, типизированная сериализация, гарантии доставки, Serverpod-интеграция и команды разработки.

Проект находится в alpha-стадии. NATS Core и JetStream являются эталонными реализациями, RabbitMQ предназначен для контролируемой production-оценки, Kafka пока имеет статус experimental.
